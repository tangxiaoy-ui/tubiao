# Tubiao

This is a custom icon library project based on [Ldesgin](https://mksmoke.ywork.me/web/lui/#/core/common)).

## About

Tubiao is an open-source icon library that provides vector (svg) files for displaying icons and symbols in digital and non-digital projects.

## Getting Started

### Prerequisites

- Node.js
- pnpm

### Installation

```bash
pnpm install
```

### Adding New Icons

1. Add your SVG files to the `icons/` directory.
2. Run `pnpm run build` to generate the packages.
